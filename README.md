# Frontend
  ReactJS with ElectronJS boiler plate to build cross-platform desktop application with modern frontend library and bigest community :)

### Usage
  
  - Clone Project
    ``` 
      git clone https://github.com/inozura/retron-boilerplate.git
    ```
  
  - Install depedencies
    ``` 
      yarn install
    ```
     or
    
    ``` 
      npm install
    ```
    
  - Run project
    ```
      yarn start
    ```
    or
    
    ``` 
    yarn install
    ```
